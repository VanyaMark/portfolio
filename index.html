Emotionary

A dynamic image management app built with Node.js, Express, and EJS. Users can add, search, filter, and delete images seamlessly. Features include real-time search, tag management via Tagify, and color extraction with color-thief-node, all designed for a smooth, intuitive user experience.

Art Tetris

A game merging art education with classic gameplay. Players align falling blocks representing artistic periods, using intuitive controls as the pace quickens. Art Tetris is both engaging and educational, familiarizing users with significant art movements.

ArtBox

A gallery management app built with JavaScript, Handlebars.js, and MongoDB. Features include an image carousel, exhibition and artist management, route-specific access, and session authentication.

DOM Cheatsheet

A visually engaging DOM cheatsheet that simplifies learning with a clear layout and thoughtful color palette. This user-friendly guide makes understanding and navigating DOM concepts easy and enjoyable.

The Rose Web

Calculator

A TDD-based project featuring robust functionality and dynamic styling. It includes comprehensive logic handling for operations and a toggle for three distinct styles, ensuring efficient data management and a seamless user experience.

Slack App

A Slack app project using Tailwind CSS to enhance its mobile view, with ongoing updates for additional media queries. Tailwind CSS allows for creative, flexible design, offering a dynamic alternative to Bootstrap and resulting in an engaging mobile interface.
